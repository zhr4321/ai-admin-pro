# Bug 排查与修复

## 适用场景

页面功能异常、编译报错、运行时错误需要 AI 辅助定位和最小化修复。

## 前置上下文

- 相关文件：@ 引用出问题的文件
- 相关规则：`project-core.mdc`

## 提示词正文

```
在 ai-admin-pro 中排查并修复以下问题：

## 现象
{描述具体现象，如：点击登录按钮无反应 / npm run build 报 TS 错误}

## 复现步骤
1. {步骤 1}
2. {步骤 2}

## 期望行为
{描述正确的预期结果}

## 约束
- 先定位根因，再给出最小 diff 修复
- 不重构无关代码
- 修复后运行 npm run build 验证
- 如涉及路由/权限，检查 src/router/index.ts
```

## 期望输出

- [ ] 根因分析清晰
- [ ] 修复 diff 最小化
- [ ] `npm run build` 通过
- [ ] 问题现象已消除
